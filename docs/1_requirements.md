# 1. 要件定義

## 1.1 プロジェクト概要

シンプルなタスク管理アプリケーション。個人のタスクをCRUD操作で管理する。

## 1.2 目的

- タスクの作成・閲覧・編集・削除
- タスクのステータス管理（未着手 / 進行中 / 完了）
- シンプルで直感的なUI

## 1.3 対象ユーザー

- 個人利用（認証なし・シングルユーザー）

## 1.4 機能要件

| ID | 要件 | 優先度 |
|----|------|--------|
| FR-001 | タスクの一覧表示 | 必須 |
| FR-002 | タスクの新規作成 | 必須 |
| FR-003 | タスクの編集 | 必須 |
| FR-004 | タスクの削除 | 必須 |
| FR-005 | タスクのステータス変更 | 必須 |
| FR-006 | ステータス別フィルタリング | 任意 |

## 1.5 非機能要件

| ID | 要件 | 詳細 |
|----|------|------|
| NFR-001 | レスポンス | API応答 500ms以内 |
| NFR-002 | 環境 | Docker Composeでローカル完結 |
| NFR-003 | データ永続化 | PostgreSQLで永続化 |

## 1.6 制約事項

- ローカル環境のみ（デプロイなし）
- 認証・認可なし
- マルチユーザー非対応
