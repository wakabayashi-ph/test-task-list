# 3. 画面設計

## 3.1 画面一覧

| # | 画面名 | パス | 説明 |
|---|--------|------|------|
| 1 | タスク一覧 | / | メイン画面。全タスクをリスト表示 |
| 2 | タスク作成 | /new | 新規タスク入力フォーム |
| 3 | タスク編集 | /edit/:id | 既存タスク編集フォーム |

---

## 3.2 画面詳細

### 画面1: タスク一覧（/）

```
┌─────────────────────────────────────────┐
│  Task List                    [+ 新規作成] │
├─────────────────────────────────────────┤
│  [全て] [未着手] [進行中] [完了]           │
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────────┐ │
│  │ □ タスクタイトル1                     │ │
│  │   ステータス: 未着手    [編集] [削除]  │ │
│  └─────────────────────────────────────┘ │
│  ┌─────────────────────────────────────┐ │
│  │ ■ タスクタイトル2                     │ │
│  │   ステータス: 完了      [編集] [削除]  │ │
│  └─────────────────────────────────────┘ │
│                                         │
│  タスクが0件の場合:                       │
│  「タスクがありません。作成してください」    │
└─────────────────────────────────────────┘
```

**操作:**
- `+ 新規作成` → 画面2へ遷移
- `編集` → 画面3へ遷移
- `削除` → 確認ダイアログ → 削除実行
- ステータスフィルタタブ → 一覧をフィルタリング

---

### 画面2: タスク作成（/new）

```
┌─────────────────────────────────────────┐
│  ← 戻る    タスク作成                     │
├─────────────────────────────────────────┤
│                                         │
│  タイトル *                              │
│  ┌─────────────────────────────────────┐ │
│  │                                     │ │
│  └─────────────────────────────────────┘ │
│                                         │
│  説明                                    │
│  ┌─────────────────────────────────────┐ │
│  │                                     │ │
│  │                                     │ │
│  └─────────────────────────────────────┘ │
│                                         │
│  ステータス                              │
│  ┌─────────────────────────────────────┐ │
│  │ 未着手                          ▼   │ │
│  └─────────────────────────────────────┘ │
│                                         │
│          [キャンセル]  [作成]             │
└─────────────────────────────────────────┘
```

**操作:**
- `作成` → タスク保存 → 一覧へ遷移
- `キャンセル` / `← 戻る` → 一覧へ遷移
- バリデーション: タイトル必須

### 画面3: タスク編集（/edit/:id）

```
┌─────────────────────────────────────────┐
│  ← 戻る    タスク編集                     │
├─────────────────────────────────────────┤
│                                         │
│  タイトル *                              │
│  ┌─────────────────────────────────────┐ │
│  │ (既存のタイトル)                     │ │
│  └─────────────────────────────────────┘ │
│                                         │
│  説明                                    │
│  ┌─────────────────────────────────────┐ │
│  │ (既存の説明)                        │ │
│  │                                     │ │
│  └─────────────────────────────────────┘ │
│                                         │
│  ステータス                              │
│  ┌─────────────────────────────────────┐ │
│  │ 進行中                          ▼   │ │
│  └─────────────────────────────────────┘ │
│                                         │
│          [キャンセル]  [更新]             │
└─────────────────────────────────────────┘
```

**操作:**
- `更新` → タスク更新 → 一覧へ遷移
- `キャンセル` / `← 戻る` → 一覧へ遷移
- バリデーション: タイトル必須
